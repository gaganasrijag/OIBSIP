# BMI Calculator

## Description
This Python program calculates the Body Mass Index (BMI) of a user based on their weight and height.

## Features
- Accepts weight in kilograms
- Accepts height in meters
- Calculates BMI
- Displays BMI category:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese

## Development Environment
- Python 3.10 (using IDLE)
