# Random Password Generator

## Description

This Python program generates a random password based on the user-specified length. It uses a combination of uppercase letters, lowercase letters, digits, and special characters to create strong passwords.

## Features

* Accepts the desired password length from the user.
* Generates random and secure passwords.
* Includes alphabets, numbers, and special characters.
* Simple command-line interface.

## Technologies Used

* Python 3.10 (using IDLE)

## How to Run

1. Open the program in IDLE.
2. Run the Python file.
3. Enter the desired password length when prompted.
4. The program will display a randomly generated password.
